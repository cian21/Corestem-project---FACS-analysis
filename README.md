# LupusBiomarker
Govermental Reseach - Lupus biomarker

바이오 의료기술개발사업인 전신홍반루푸스 줄기세포 치료제의 유효성 검증을 위한 임상 바이오마커 발굴 및 실용화의 최종보고서 작성을 목적으로 하였습니다.
해당 README 파일의 작성은 2단계를 작성하고 있으며, 연구기간은 2019년 1월 ~ 2021년 3월입니다.

2단계의 연구 범위는 RNA-seq 분석을 통한 바이오마커 모듈 검증 및 보완이며,

1. 임상1상, 응급임상, IIT의 MSC 치료제 투여 전, 후에서 얻은 혈액에서 PBMC를 분리하여 RNA-seq 진행합니다.
2. NCBI의 GEO에 공개된 루푸스 관련 데이터(case-control, MSC, mouse, SNP, etc.)를 분석하여 공개 DB 구축, 임상 데이터 분석을 기반으로 한 자체 DB 구축합니다.
3. 1단계의 샘플 수를 임상2상 샘플로 보완하여 1단계에서 생성하였던 모든 바이오마커 모듈 검증 및 보완합니다.
4. 루푸스 신염의 진행 예측, MSC의 치료효과 예측, MSC치료에 대한 반응자/비반응자 예측 바이오마커를 찾고자 합니다.
5. 공개 오믹스 분석을 기반한 통합 루푸스 DB 구축, PBMC RNA-seq 분석, Differential Expression (DE) analysis, Modular repertoire analysis, Weighted Gene Correlation Network Analysis (WGCNA), Gene set / pathway enrichment, ontology analysis을 진행합니다.
6. 임상1상, 응급임상, IIT 샘플 데이터를 이용하여 자체 루푸스 DB 증설하며, 샘플을 추가분석하여 1단계에서의 후보 바이오마커 모듈 검증 및 보완하고자 합니다.
